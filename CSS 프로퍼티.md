## CSS 프로퍼티

#### 1. 키워드

각 프로퍼티에 따라 사용할 수 있는 키워드가 존재한다.



#### 2. 크기 단위

대표적인 크기 단위는 px, em, %이다.

px는 절대값이고, em, %는 상대값이다.

대부분 브라우저의 폰트 사이즈의 기본값은 16px, 1em, 100%이다.



#### 3. 색상 표현 단위

색상을 지정하기 위해서는 키워드를 사용한다.

사용이 간편하나 표현할 수 있는 색상의 수는 제한된다.

**다양한 색상을 표현하기 위해 색상 표현 단위를 사용할 수 있다.**
